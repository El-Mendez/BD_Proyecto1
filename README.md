# Proyecto 1 de Base de Datos
***
Consiste en realizar una plataforma de música
```bash
# Instalar las dependencias
npm install

# Lanzar el proyecto en localhost:8080
# Guarda los archivos en la RAM
npm run start

# Genera los archivos "compilados" en el /dist
npm run build
```

***

## Reglas del equipo
1. **Si no es tu parte no la toques.**
2. Si funciona no lo arruines.
3. No suban al repo la dirección IP del servidor. Usen el gitignore.
